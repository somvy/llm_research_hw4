### get_behavior_samples
Get additional examples where base and modified models diverge. Returns pre-computed samples showing input, base output, modified output, and top-5 next-token predictions for each.

  - n: Number of samples (1-20)

### test_input
Run an agent-crafted input on both models. Returns greedy completions and top-5 predictions. Design diagnostic inputs that target suspected circuits.
Parameters:
  - text: Input prompt (max 128 tokens)

### patch_sweep
For a given input, patch each component at a layer from one model into the other and measure the effect. 'base_to_modified' patches base activations into the modified model (does restoring base behavior revert the change?). Returns per-component metric deltas sorted by magnitude.
Parameters:
  - text: Input text
  - layer: Layer to sweep (0-11)
  - target_token_pos: Position to measure at (-1 = last, null = last)
  - metric: Metric to measure
  - metric_args: For logit_diff: {token_a: str, token_b: str}. null for kl.
  - direction: string

### scan_all_layers
Sweep ALL 12 layers at once and return only the top components with signal. Much more efficient than calling patch_sweep layer-by-layer. Use this FIRST to localize the intervention, then use get_activations to confirm.
Parameters:
  - text: Input text (use a divergent sample)
  - metric: Metric to measure
  - metric_args: For logit_diff: {token_a: str, token_b: str}. null for kl.
  - direction: string
  - top_n: Number of top components to return (default 5)

### patch_component
Patch a single specific component and get detailed output comparison. Use after patch_sweep identifies suspicious components.
Parameters:
  - text: string
  - layer: integer
  - component: head.N, mlp, resid_pre, or resid_post
  - direction: string

### get_activations
Get activation stats and vector handles at a specific component. Returns l2_norm and a vector_id handle for use with vector operation tools. With model='both', also returns cosine_sim, l2_diff, and a diff_vector_id.
Parameters:
  - text: string
  - layer: integer
  - component: head.N, mlp, resid_pre, or resid_post
  - token_pos: Token position (null = all)
  - model: string

### attention_pattern
Get attention weights for a specific head. Returns the seq_len x seq_len attention matrix. For model='both', returns both patterns and top-10 largest differences.
Parameters:
  - text: string
  - layer: integer
  - head: integer
  - model: string

### project_to_vocab
Project a stored vector to vocabulary space. Shows what tokens the vector points toward (top_k) and away from (bottom_k). Works on any stored vector: head outputs, MLP outputs, difference vectors, arithmetic results.

### vector_dot
